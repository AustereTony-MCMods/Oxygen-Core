+ Добавлены параметры конфигурации:
 Клиент [Core]
 - "notifications_menu_key" [int] - идентификатор клавиши меню уведомлений
 - "settings_menu_key" [int] - идентификатор клавиши меню настроек
 - "privileges_menu_key" [int] - идентификатор клавиши меню привилегий
 - "accept_key" [int] - идентификатор клавиши принятия запроса
 - "reject_key" [int] - идентификатор клавиши отмены запроса
 - "interaction_key" [int] - идентификатор клавиши взаимодействия

 - "advanced_logging" [boolean] - если "true", будет активировано расширенное логирование всех событий Oxygen
 
+ Добавлены команды:
 Сервер - Операторы [/oxygenop]
 - Выполнить скрипт из чата: /oxygenop core -script -chat <script text>

0.10.0b [2020-01-16]
+ Доработана система привилегий. Система групп заменена ролями, игрок может иметь до пяти ролей.
Роль определяет приаилегии (разрешения), доступные игроку и форматирование чата (если активно) - префикс и цвета.
По умолчанию игроки не имеют ролей, их разрешения определяются конфигурацией. Для просмотра ролей игрока и управления привилегиями 
(для операторов) добавлено специальное меню. Вызвать меню можно командой /oxygenc core -privileges. Так же его можно открыть через 
общее меню (Oxygen Menu), если разрешить через меню настроек клиента его добавление в общее меню.
+ Добавлены привилегии:
 - "core:exposeOfflinePlayers" [id - 10, type - BOOLEAN] - возможность видеть игроков онлайн со статусом активности 
   "не в сети" и взаимодействовать с ними (в меню).

+ Доработана система виртуальной валюты, oxygen теперь поддерживает несколько виртуальных валют единовременно. Основной валютой 
является oxygen coins [индекс 0], дополнительные валюты могут быть использованы только в некоторых модулях (например Merchants).
Добавлено две новых валюты: shards [индекс 1] и vauchers [индекс 2]. Существует возможность кастомизации виджетов валюты (иконки и названия).
+ Изменён принцип работы провайдеров валют (currency providers). Провайдер теперь может быть использован для замены существующей валюты или
добавления новой (валюта зависит от конкретного мода, провайдер для которого использован). Несколько провайдеров могут быть использованы 
единовременно. Индекс валюты провайдера настраивается через конфиг провайдера, по умолчанию любой провайдер заменяет основную валюту oxygen 
coins [индекс 0]. Для провайдеров доступна опция принудительной синхронизации баланса, котороая должна использоваться если мод, добавляющий валюту
выступает в роли API для других модификаций (пример: использование Grand Economy для взаимодействия с экономикой Sponge).
+ Доработаны команды чата. Oxygen добавляет три команды, для которых другие модули регистрируют свои аргументы: /oxygenc (доступна только для клиента), 
/oxygens (доступна для любого игрока, используется для серверных операций), /oxygenop (доступна для операторов (permission level 2) и консоли). 
Команды, принимающие в качестве аргумента ник игрока поддерживают селекторы (target selectors).
+ Обновлены параметры конфигурации:
 Клиент [Core]
 - "data_save_peroid_seconds" [int] - период сохранения данных в секундах
 - "enable_notifications_key" [boolean] - разрешить добавление клавиши "открыть меню уведоммлений"
 - "enable_accept_key" [boolean] - разрешить добавление клавиши "принять запрос"
 - "enable_reject_key" [boolean] - разрешить добавление клавиши "отклонить запрос"
 - "enable_interaction_key" [boolean] - разрешить добавление клавиши "взаимодействие"
 - "date_format_pattern" [string] - паттерн формата отображения даты
 
 Сервер [Core]
 - "io_threads_amount" [int] - количество потоков для обработки операций ввода-вывода
 - "routine_threads_amount" [int] - количество потоков для обработки рутинных операций
 - "scheduler_threads_amount" [int] - количество потоков для обработки отложенных задач
 - "data_save_period_seconds" [int] - период сохранения данных в секундах
 - "sync_configs" [boolean] - разрешить синхронизацию помеченных опций конфига с клиентами
 - "enable_privileges" [boolean](синхронизируется) - разрешить использование системы привилегий

 - "sync_entities_absorption_value" [boolean] - разрешить синхрониацию данных о значении поглощения отслеживаемых сущностей
 - "sync_entities_armor_value" [boolean] - разрешить синхрониацию данных о запасе брони отслеживаемых сущностей
 - "sync_entities_active_effects" [boolean] - разрешить синхрониацию данных об активных эффектах отслеживаемых сущностей
 
 Сервер [Privileges]
 - "enable_formatted_chat" [boolean](синхронизируется) - разрешить форматирование чата (роли)
 - "enable_chat_formatting_role_management" [boolean](синхронизируется) - разрешить смену активного режима форматирования чата для игрока (при наличии ролей)
 - "enable_custom_formatted_chat" [boolean] - разрешить использование кастомного паттерна для форматирования чата
 - "custom_formatted_chat_pattern" [string] - паттерна для форматирования чата (если у игрока нет префикса)
 - "custom_formatted_chat_prefix_pattern" [string] - паттерна для форматирования чата (если у игрока есть префикс)
 - "custom_formatted_default_chat_color" [int] - индекс цвета чата по умолчанию (индекс элемента TextFormatting)
 - "show_prefix" [boolean] - показывать заданный ролью префикс
 - "enable_nickname_color" [boolean] - использовать заданный ролью цвет ника
 - "enable_prefix_color" [boolean] - использовать заданный ролью цвет префикса
 - "enable_chat_color" [boolean] - использовать заданный ролью цвет сообщения
 
+ Добавлены команды:
 Сервер - Операторы [/oxygenop]
 - Добавление роли игроку (privileges): /oxygenop core -privileges -add-role <target> <role id>
 - Удаление роли игрока (privileges): /oxygenop core -privileges -remove-role <target> <role id>

 - Проверка баланса (игрока): /oxygenop core -currency -balance <target> <currency index>
 - Установка баланса: /oxygenop core -currency -set <target> <currency index> <value>
 - Пополнение баланса: /oxygenop core -currency -add <target> <currency index> <value>
 - Уменьшение баланса: /oxygenop core -currency -remove <target> <currency index> <value>

 - Отправка уведомления (для меню уведомлений): /oxygenop core -notification <target> <message>
 
 Клиент [/oxygenc]
 - Открыть меню привилегий (privileges): /oxygenc core -privileges
 - Проверка значения привилегии (privileges): /oxygenc core -privileges -get-value <privilege id>
 
 - Проверка баланса (игрока): /oxygenc core -currency -balance <currency index>
 
 - Открыть меню уведомлений: /oxygenc core -notifictions
 
 - Открыть меню настроек клиента: /oxygenc core -settings
 - Перезагрузить файл настроек: /oxygenc core -settings -reload-settings
 - Перезагрузить 'свойства' (система кастомизации элементов, например виджетов виртуальной валюты): /oxygenc core -settings -reload-properties
 
 - Принять запрос: /oxygenc core -request -accept
 - Отклонить запрос: /oxygenc core -request -reject

+ Изменена система загрузки конфигурации, а так же обновлены опции конфигурации. Рекомендуется удалить папку конфигурации oxygen полностью перед
запуском новой версии.
+ Добавлена основа системы каналов для чата. На данный момент используется модулем Groups для добавления чата группы.
+ Добавлена система отслеживания параметров сущностей. На данный момент система отслеживает здоровье, поглощение, броню и активные эффекты
ближайших игроков, а так же помеченных игроков. Используется модулем Groups для отображения данных участников группы в оверлэе.
+ Добавлена система фильтрации пакетов, отправляемых игроками на сервер. Используется для ограничения обработки пакетов, посылаемых слишком часто (например смена статуса 
активности возможна не чаще 1 раза в секунду). Кулдаун установлен для всех отправляемых запросов, для большенства операций он равен 1 секунде.
+ Добавлена система таймаутов. Таймауты используются для фильтрации операций игроков, которые не должны выполнятся без определённого условия.
Пример: отправка запросов на покупку товара у торгоцев (Merchants) будет возможна только в течении определённого времени после авторизованного отрытия (командой) меню 
для конкретного игрока.
+ Пресет категорий предметов (используемый в меню торговли модуля Trade) перенесён в Core. Категории сгенерированы на основе
вкладок креативного инвентаря ванильной версии игры и не изменяются динамически. Категории можно кастомизировать.
+ Добавлено меню настроек клиента. Вызвать меню можно командой /oxygenc core -settings. Так же его можно открыть через общее меню (Oxygen Menu), если разрешить через меню настроек клиента 
его добавление в общее меню. Меню настроек содержит два раздела: общие настройки и настройки интерфейса. Модули oxygen будут добавлять свои настройки в это меню.
